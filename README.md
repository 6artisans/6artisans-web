# 6artisans web based on Middleman platform

## Middleman
- https://middlemanapp.com/

## Middleman docs
- https://middlemanapp.com/basics/install/

## How to run
- first you need `ruby 2.3.0` (use RVM or rbenv)
- run `bundle`
- start middleman server `middleman`

# How to deploy
`rake site:publish`
